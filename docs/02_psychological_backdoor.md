## 1. The "API" Loophole (The Leaky Pipeline)

**The Myth:** "We do not sell your data."
**The Reality:** They don't have to _sell_ it; they grant _access_ to it via APIs (Application Programming Interfaces).

- **The Mechanism:** Platforms (Facebook, TikTok, etc.) allow third-party developers to plug into their user base via APIs to "enhance user experience."
- **The Exploit:** These third parties (Data Brokers) use this access to scrape behavioral metadata—not just _who_ you are, but _how_ you behave (scroll speed, pause time, click frequency).
- **The Receipt:**
  - **Cambridge Analytica (The Prototype):** They didn't "hack" Facebook. They used a legitimate API permission to harvest data from 87 million users via a simple "personality quiz."
  - **Current Status (2025):** While the front door was "closed" (permissions tightened), the **Server-to-Server (CAPI)** integrations allow advertisers to send offline customer data back to Facebook/Google to match profiles, effectively bypassing browser-based privacy tools.

**Objective:** Maximizing "Time on Device" (TOD) by bypassing executive function.
**The Method:** Variable Ratio Reinforcement Schedule (The Slot Machine).

- **The Logic:** If you get a reward (a generic "Like") every time, you get bored. If you get a reward _randomly_, you become obsessive.
- **The Feature:** **"Infinite Scroll"** (Aza Raskin, the inventor, has publicly regretted this).
  - **Mechanism:** It removes the "stopping cue" (the bottom of the page). The brain never gets the signal to reassess: "Should I continue?"
  - **The Result:** "Dopamine-Scrolling." A trance state where the user is looking for the "winning spin" (the one video that hits the spot).
- **The Patent Trail:**
